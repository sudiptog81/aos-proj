{\footnotesize\ttfamily int stat\+\_\+file (\begin{DoxyParamCaption}\item[{const char $\ast$}]{path }\end{DoxyParamCaption})}



Queries information about a file using {\ttfamily stat()} system call. 

It prints the following information about the file to the console\+:
\begin{DoxyItemize}
\item File type
\item Device ID
\item Major device ID
\item Minor device ID
\item File size (in bytes)
\item Blocks allocated
\item Filesystem block size
\item Inode
\item Links
\item Owner permissions
\item Group permissions
\item Other permissions
\item Owner user and UID
\item Owner group and GID
\item Last accessed
\item Last modified
\item Last changed
\end{DoxyItemize}

This function is invoked by the {\ttfamily info} command.
