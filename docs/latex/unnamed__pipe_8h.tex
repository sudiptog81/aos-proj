Uses unnamed pipes for message passing between a parent and a child process. 

It creates a pipe using the {\ttfamily pipe()} system call and forks a child process. The parent process writes to the pipe and the child process reads from it.

The user is prompted to enter a message to write to the pipe. The user can exit the program by entering {\ttfamily \+:q} as the message.

This function is invoked by the {\ttfamily pipe} command.

\begin{DoxyReturn}{Returns}
0 on success, -\/1 on failure 
\end{DoxyReturn}
