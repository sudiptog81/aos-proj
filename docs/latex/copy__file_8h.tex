Copies a regular file using an unnamed pipe and the {\ttfamily read()} and {\ttfamily write()} system calls. 

It copies the content of the source file specified by {\ttfamily src} to the target specified by the {\ttfamily dest} parameter.

This function is invoked by the {\ttfamily copy} command.

In case the target file already exists, the user is prompted for confirmation and needs to pass the {\ttfamily -\/f} flag to force this operation.
