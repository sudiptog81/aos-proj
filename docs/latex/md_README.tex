This is a command-\/line program that can be used to demonstrate system calls in UNIX. It is written in C and uses the GNU C Library and the POSIX API.\hypertarget{md_README_autotoc_md1}{}\doxysection{Features}\label{md_README_autotoc_md1}
This program supports the following features as of now\+:


\begin{DoxyItemize}
\item creating a file/named pipe with permissions as given by user
\item reading from a file and printing the contents to stdout
\item writing to a file from stdin
\item displaying statistical information about a file
\item copying the contents of one file to another using an unnamed pipe
\item using a named pipe to communicate between two processes
\end{DoxyItemize}

{\bfseries{Note}}\+: The reading and writing operations also support offsets and number of bytes to be read or written.\hypertarget{md_README_autotoc_md2}{}\doxysection{Commands}\label{md_README_autotoc_md2}
