Creates a regular file or a named pipe using the {\ttfamily creat()} or {\ttfamily mknod()} system call. 

It also sets the file permissions to {\ttfamily perms} by using the mode bits supplied.

This function is invoked by the {\ttfamily create} command.

In case the target file already exists, the user is prompted for confirmation and needs to pass the {\ttfamily -\/f} flag to force this operation.
